Spoj EDIST solution

By Einstein | October 8, 2015
0 Comment

Spoj EDIST solution. Spoj Edit Distance solution.

Category: Dynamic Programming

This question is the direct application of the Edit Distance Problem , detailed explanation along with working code has been provided in the linked post, read it to understand how it works.

Here is the working code:

#include<bits/stdc++.h>
using namespace std;
/*An extra row and column has been 
   included to take care of the cases when
   either of the sritng is of length 0*/
int E[2004][2004];

int EditDistance( string A , string B , int n ,int m){
   
   int i,j;
   
   for( i=0;i<=n;i++){
      for(j=0;j<=m;j++){
         /*String A is empty,
         Only way is to Insert everything*/
         if( i==0)
            E[i][j]=j; 
         /*String B is empty,
         Only way is to Delete everything*/	
         else if(j==0)
            E[i][j]=i; 
         /*If the last character is same*/	
         else if( A[i-1]==B[j-1])
            E[i][j] = E[i-1][j-1];
         /*If the last character is different*/	
         else
         {
            E[i][j] = min( E[i][j-1],min( E[i-1][j], E[i-1][j-1]))+1;
         }	
      }
   }
   return E[n][m];
}

int main() {
   string A,B;
   int t;
   scanf("%d",&t);
   while(t--)
   {
      cin>>A>>B;
      printf("%d\n",EditDistance( A , B , A.length(),B.length()));
   }
   return 0;
}
